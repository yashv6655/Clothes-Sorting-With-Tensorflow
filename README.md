# Clothes-Sorting-With-Tensorflow

## About
This is a machine learning project built using TensorFlow 2.0.

This model sorts images of clothes based on different categories:
- T-Shirt
- Shorts
- Hoodie
- Coat
- Sandal
- Shirt
- Shoes
- Bag
- Boots


## Technologies
Buit Using:
- TensorFlow
- Numpy
- Matplotlib
